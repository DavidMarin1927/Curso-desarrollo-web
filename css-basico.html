Definir estilos del HTML con CSS

CSS (Cascade Style Sheets) se usa para controlar el formato visual de una página.

Con CSS se puede controlar el color, fuente, tamaño del texto, espaciado, alineamiento, imágenes de fondo, cómo se ve una página dependiendo del tamaño de pantalla del dispositivo que estemos usando, etc.

Formas de usar el CSS

  • Inline - Usando el atributo style dentro del elemento HTML.
  • Internal - Usando la etiqueta <style> dentro de la sección del <head>
  • External - Usando un archivo CSS externo.

Inline CSS

Se usa para aplicar un estilo a un elemento HTML en concreto mediante el atributo style dentro de la etiqueta del elemento. Ejemplo de un título con la letra azul:

Internal CSS

Se usa para aplicar estilo a un documento en concrecto, y define dentro en una etiqueta >Style dentro de la cesion head del documento ref en linea

Extrenal CSS

Es un archivo con la extecion, css que contiene exclusivamente codigo en, CSS. se puede aplicar al documento mendiante el elemento <link

Las dos principales ventajas de externalizar al CSS son :

  • los estilos se pueden aplicar a tantos documentos como queramos
  • la longitud del codigo disminuye sensiblemente
