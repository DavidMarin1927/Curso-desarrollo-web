Atributos de elementos HTML

Los atributos proveen información adicional de elementos HTML.

  • Todos los elementos HTML pueden tener atributos.
  • Los atributos proveen info adicional de los elementos.
  • Los atributos se especifican en la etiqueta de apertura
  • Los atributos normalmente tienen estructura de nombre="valor"

Ejemplo. Un link se define con la etiqueta <a>. El atributo "href" especifica la URL a la página a la que irá el link.
