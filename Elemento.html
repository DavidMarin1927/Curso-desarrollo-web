Links o hiperviculos

Los links se encuentras en todo tipo de paginas. permiten al usuario navegar por los documentos de la web.

cuando clicas en un link te mueve a otro documento o a una parte concreta del mismo.

cuando pasa el cursor sobre el link, este se torna en una mano.

Un link se define mediante el elemento <a <

La sintaxis de un link, por norma general, es:
<a href="url"лTexto del link </a<

El atributo mas importante de un link es el "href" ya que es este el que indica el punto al que nos llevara el link.

Los links tienen 3 estados:

  • Son azules cuando no han sido visitados
  • Morados cuando ya han sido clicados
  • y rojos cuando estan ativos.

Lon links se pueden modificar mendiante css para que tengan un aspecto completamente distinto.
